# Anthropic Academy

<Callout type="info">
공식 사이트: [anthropic.com/learn](https://www.anthropic.com/learn)
</Callout>

Claude를 만든 Anthropic이 직접 운영하는 교육 플랫폼입니다.
API 개발부터 실무 활용까지, 단계별로 구성된 코스를 무료로 수강할 수 있고 완료 시 인증서를 받을 수 있습니다.

---

## 주요 코스

### Claude 101

Claude의 기본 개념과 사용법을 배우는 입문 코스입니다.

- AI와 대화하는 기본 원리
- Claude의 강점과 한계 이해
- 다양한 활용 시나리오 체험

### Claude Code in Action

Claude의 코드 기능을 실습 중심으로 배우는 과정입니다.

- Claude Code 설치와 기본 사용법
- 실제 프로젝트에서 Claude Code 활용
- CLAUDE.md를 통한 프로젝트 컨텍스트 관리

### 추가 코스

- **AI Fluency** — AI 리터러시를 높이는 기초 과정
- **Model Context Protocol (MCP)** — 외부 도구와 Claude를 연결하는 프로토콜 학습

---

## 학습 경로

Anthropic Academy는 세 가지 학습 경로를 제시합니다:

| 경로 | 대상 | 핵심 내용 |
|------|------|-----------|
| **Build with Claude** | 개발자 | API 개발, SDK 활용, 프롬프트 엔지니어링 |
| **Claude for Work** | 실무자·팀 | 조직 내 도입 전략, 생산성 향상 |
| **Claude for Personal** | 개인 사용자 | 일상·개인 프로젝트 활용 |

---

## 워크숍과의 연결

이 워크숍의 **세션 1(만든다)** 에서 AI 도구를 처음 써본 분이라면,
Anthropic Academy의 <strong>Claude 101</strong>로 기초를 다지고,
<strong>Claude Code in Action</strong>으로 **세션 3(자산화)** 에서 배운 내용을 심화할 수 있습니다.
